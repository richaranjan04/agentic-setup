# AI Search Admin Console — Demo

Interactive demo of an AI Search admin experience with Otto, an AI assistant that guides admins through configuring search sources, validating results, and troubleshooting issues.

## Run locally

Open `AIS_Agentic_Demo.html` directly in a browser, or serve the folder with any static server.

## Deploy to GitHub Pages

1. Push this repo to a new GitHub repository.
2. In the repository settings, go to **Pages**.
3. Select **Deploy from a branch**, choose `main`, and set the folder to `/` (root).
4. GitHub Pages will serve `AIS_Agentic_Demo.html` at `https://<your-username>.github.io/<repo-name>/AIS_Agentic_Demo.html`.

If you want the page to load at the repo root, rename `AIS_Agentic_Demo.html` to `index.html`.
